# Test Projekt - Next.js Anwendung

Dies ist ein [Next.js](https://nextjs.org) Projekt, das mit `create-next-app` erstellt wurde. Es dient als Grundlage für moderne Webanwendungen mit TypeScript-Unterstützung.

## 🚀 Funktionen
- Moderne Next.js App Router Architektur
- Vollständige TypeScript-Integration
- Optimierte Schriftarten mit `next/font`
- Schnelle Entwicklung mit Hot Reload

## 🛠️ Verwendete Technologien
- **Next.js 15** - React Framework
- **TypeScript** - Typisierte Programmiersprache
- **Tailwind CSS** - CSS-Framework
- **ESLint** - Code-Qualitätssicherung
